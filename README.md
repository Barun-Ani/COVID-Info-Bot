# COVID-Info-Bot
## Industry*:
Health

## Project Title*:
COVID19 Info Chatbot

## Problem Statement/Opportunity*:
Building a Chatbot to Address Covid-19-Related Concerns.COVID 19 is one the most major concern for countries. Lack of credible information is a very concerning thing that we all must deal with.

## Project Description*:
This is a chatbot that will give answers to most of your corona related questions/FAQ. The chatbot will give you answers from the data given by WHO(https://www.who.int/).
